# meander-channel-cutoffs
This repository contains MATLAB codes for plotting the figures in the manuscript “Universal geometric rules govern meandering channel cutoffs” by Wu et al.

Figures 2 and 3 can be reproduced by executing the scripts compileAllSystems.m and plotCutoffStat.m.

Modeling results shown in Figure 4 can be reproduced by executing the script plotModelResults3.m.
